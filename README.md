## 開発コマンド

```
// ローカル開発環境構築
// to http://localhost:3000/
npm run dev

// ビルド
npm run build
// lint
npm run lint
// lint -fix
npm run format
// テスト実行
npm run test
```

## テスト概要

- DOMはdata-testidで指定すること
- デプロイ時はdata-testidは削除される
